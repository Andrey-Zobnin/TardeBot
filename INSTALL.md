# Инструкция по установке и запуску TardeBot

## Требования

- Python 3.8 или выше
- pip (менеджер пакетов Python)
- Интернет-соединение для получения данных о акциях

## Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/Andrey-Zobnin/TardeBot.git
cd TardeBot
```

### 2. Создание виртуального окружения (рекомендуется)

```bash
python -m venv venv

# Активация на Windows
venv\Scripts\activate

# Активация на Linux/Mac
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка конфигурации

Скопируйте файл `.env.example` в `.env` и заполните необходимые параметры:

```bash
cp .env.example .env
```

Отредактируйте файл `.env`:
- `ALPHA_VANTAGE_API_KEY` - получите бесплатный API ключ на https://www.alphavantage.co/
- `DEFAULT_SYMBOL` - символ акции для торговли (по умолчанию AAPL)
- `INITIAL_BALANCE` - начальный баланс для торговли

## Запуск

### Демонстрационный режим

Для тестирования функциональности без реальной торговли:

```bash
python demo.py
```

### Полный запуск бота

```bash
python src/main.py
```

### Запуск тестов

```bash
pytest tests/
```

## Важные замечания

⚠️ **ВНИМАНИЕ**: Этот бот создан в учебных целях. Не используйте его для реальной торговли без тщательного тестирования и понимания рисков.

- Бот работает с демонстрационными данными
- Для реальной торговли требуется интеграция с брокерским API
- Всегда тестируйте стратегии на исторических данных перед использованием

## Получение API ключей

### Alpha Vantage (бесплатно)
1. Перейдите на https://www.alphavantage.co/support/#api-key
2. Зарегистрируйтесь и получите бесплатный API ключ
3. Добавьте ключ в файл `.env`

## Устранение неполадок

### Ошибки установки зависимостей
```bash
pip install --upgrade pip
pip install -r requirements.txt --no-cache-dir
```

### Проблемы с TensorFlow
```bash
pip install tensorflow-cpu  # Если нет GPU
```

### Проблемы с получением данных
- Проверьте интернет-соединение
- Убедитесь, что API ключ корректен
- Проверьте лимиты API (Alpha Vantage: 5 запросов в минуту для бесплатного плана)
