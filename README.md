# TardeBot - Алгоритмический Трейдинг Бот

Автоматический бот для торговли акциями на основе машинного обучения.

## Описание проекта

Бот анализирует цены акций и использует модель машинного обучения для предсказания будущих цен. На основе прогнозов принимает решения о покупке или продаже.

## Основные функции

- Получение текущих и исторических цен акций
- Предсказание будущих цен с помощью ML модели
- Автоматическое принятие торговых решений
- Логирование всех операций

## Структура проекта

```
TardeBot/
├── src/
│   ├── data/           # Модули для работы с данными
│   ├── models/         # ML модели
│   ├── trading/        # Торговая логика
│   └── utils/          # Вспомогательные функции
├── tests/              # Тесты
├── config/             # Конфигурационные файлы
└── requirements.txt    # Зависимости
```

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

